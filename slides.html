Galaxy - Bioinformatics in a Browser üåå

  • What is Galaxy? Web-based platform requiring zero installation ‚Äì analyse data from anywhere with just your browser
  • Why Start Here? Visual workflow builder, automatic parameter documentation, and built-in tutorials make complex analyses approachable
  • Our Workflow: Upload FASTQ files ‚Üí Quality control (FastQC, NanoPlot) ‚Üí Filter reads (Porechop, Fastp) ‚Üí Identify pathogens (Kraken2) ‚Üí Visualise results (Krona)
  • The Power: Click-based interface backed by the same robust tools used in command-line analysis ‚Äì simplicity without sacrifice

Galaxy Workflow - From Reads to Results üìä

  • Step 1 - Data Upload: Import your Barcode10 and Barcode11 samples directly from Zenodo into your Galaxy workspace
  • Step 2 - Quality Assessment: FastQC and MultiQC generate comprehensive reports showing read length, quality scores, and potential issues
  • Step 3 - Read Processing: Porechop removes adapter sequences, Fastp filters low-quality reads ‚Äì typically retaining 70%+ of good data
  • Step 4 - Pathogen ID: Kraken2 with BabyKraken database identifies Salmonella species in minutes; Krona creates interactive visualisation charts

Galaxy Advantages - Learning by Doing ‚úÖ

  • Reproducibility: Every analysis automatically saved as a shareable workflow you can rerun or publish
  • Transparency: Each tool shows exactly what parameters were used, eliminating "black box" syndrome
  • Community Support: Galaxy Training Network tutorials guide you step-by-step through pathogen detection workflows
  • Transition Ready: Skills learned in Galaxy translate directly to understanding command-line equivalents

Command-Line Interface - Unleashing Computational Power üíª

  • Why CLI? Process hundreds of samples with a single command, automate repetitive tasks, and access the full power of bioinformatics tools
  • The Learning Curve: Initially steeper than Galaxy, but exponentially more powerful for large-scale analyses and custom workflows
  • Your Foundation: Linux basics (navigating directories, creating files, using pipes), Conda environments for tool management
  • The Payoff: Reproducible scripts, parallel processing, integration with high-performance computing clusters

CLI Workflow - The Same Analysis, Supercharged ‚ö°

  • Environment Setup: conda activate pathogen_detection loads all required tools (FastQC, Kraken2, Fastp, etc.) in one isolated environment
  • Parallel Processing: parallel command runs FastQC on all samples simultaneously ‚Äì what takes 20 minutes sequentially finishes in 2
  • Scripting Power: Create reads_paths.tab file once, then pipe it through FastQC ‚Üí Porechop ‚Üí Fastp ‚Üí Kraken2 automatically

CLI Hands-On - Taxonomic Profiling üî¨

  • BabyKraken Database: Lightweight 11 MB database optimised for bacterial pathogens ‚Äì perfect for learning and rapid screening
  • Kraken2 Classification: Assigns taxonomic labels using exact k-mer matches, processing thousands of reads per second
  • Krona Visualisation: Interactive hierarchical pie charts let you explore results from domain ‚Üí phylum ‚Üí genus ‚Üí species
  • Standard Database Option: For comprehensive analyses, upgrade to 16 GB standard database capturing broader microbial diversity

CLI Advantages - Professional Workflows üèÜ

  • Reproducibility: Save every command in a script ‚Äì share with colleagues, rerun months later, publish in methods sections
  • Scalability: Analyse 2 samples or 200 samples with identical effort; just add more compute resources
  • Flexibility: Customise parameters, chain tools in novel ways, integrate custom scripts ‚Äì you're in complete control
  • Industry Standard: Most genomic surveillance labs, reference facilities, and research groups use CLI-based pipelines
